Optimized Product Quantization for Approximate Nearest Neighbor Search

Overview

Space decomposition has been shown to have great impact on the search accuracy in Product-Quantiztion. In this paper, we optimize product quantization by minimizing quantization distortions with respect to the space decomposition and the quantization codebooks.

  Product quantization involves decomposing the D-dimensional vector space into M subspaces, and computing a sub-codebook for each subspace. M is determined by the budget constraint of memory space and the computational cost, and is predertermined in practice. We use an orthonnormal matrix R to represent the space decomposition by which the D-dimensional vector space is first transformed.

A Non-Parameter Solution

基本思想：无需假设数据分布，交替优化空间分解矩阵 R 和子码本
算法步骤：

  1. 随机初始化 R、子码本Cm{C^{m}}Cm、各个向量的索引im{i^{m}}im
  2. 固定 R 优化码本：将数据投影到 R 空间，对每个子空间独立运行 k-means 聚类，更新子码本
  3. 固定码本优化 R：将数据投影到当前子码本空间，通过正交普鲁克问题(Orthogonal Procrustes problem) 求解最优 R
  4. 重复步骤2-3，直到算法收敛

A Parametric Solution

Our parametric solution first computes the covariance matrix ∑\sum∑ of the data and use Eigenvalue Allocation to generate the orthonnormal matrix R,which determines the space decomposition. The data are then transformed by R. The original PQ algorithm is then performed on these transformed data.

基本思想：假设数据服从高斯分布，通过特征值分配平衡子空间的方差，最小化理论失真下界。
算法步骤：

  1. PCA对齐：对数据进行PCA变换，使各个维度独立
  2. 特征值排序：将特征值降序排序
  3. 特征值分配：将特征值依次分配到 M 个子空间，每个子空间包含D/MD/MD/M个特征值，确保各子空间的特征值乘积均衡(Sequentially pick out the largest eigenvalue and allocate it to the bucket having the minimum product of the eigenvalues in it unless the bucket is full)
  4. 生成空间分解矩阵 R：根据分配的特征值对应的主成分方向构建 R(The principle directions corresponding to the eigenvalues in each bucket form the subspace)
