FilterDiskann, Graph Algorithms for Approximate Nearest Neighbor Search with Filters

Overview

  As ANNS-based dense retrievl becomes ubiquitous for search and recommendation scenarios, efficiently answering filtered ANNS queries has become a critical requirement, which asks for the nearest neighbors of a query’s embefing from the points in the index that match the query’s labels such as date, price range, language. We present two algorithms with native support for faster and more accurate filtered ANNS queries: one with streaming support and another based on batch construction. Central to our algorithms is the construction of a graph-structured index which forms constructions not only based on the geomotry of the vector data, but also the associated label set.

  A comman thread across most current methods for filtered ANNS is that they only modify the search step, and not the index build step, which we argue is sub-optimal:

  • Post-processing: build a standard ANNS index, query as usual, and post-process the results by selecting only those results returned by the index that match the query filter.[ easy to implement, but performs poorly since we may have to retrieve a very large number of candidates from index]

  • Inline-processing: apply the post-processing approach “on the fly”. By including filter metadata with each entry in the inverted index, an inline-processing search can skip points in the clusters do not match the filters of the query.[ Pinecon为经典代表 ]

Algorithms

  Our main contricutions are two simple-yet-effective graph based algorithms for filtered ANNS - FilteredVamana and StichedVamana - that build upon the Vamana graph. In this work, we limit to simple filters - exact match with one filter, we can easily extend this to disjunction(OR) of serveral filters by simply fidning the answer corresponding to each individual filter, and aggretating and sorting these results by distance from the query. We leave the case of conjunctions(AND) of several filters and other more complicated expressions as a challenging avenue for future work.[ 并集多查询 ]
